# Learning Paths for Abacus Learn

## Grade K (Kindergarten)

### Path 1: Counting Fundamentals
- **Description:** Master counting from 1-20 and number recognition
- **Estimated Time:** 25 minutes
- **Modules:**
  * k-counting-1-20
  * k-number-order
  * k-comparing-sizes

### Path 2: Shape Discovery
- **Description:** Learn to identify and describe basic shapes
- **Estimated Time:** 20 minutes
- **Modules:**
  * k-shapes-basic
  * k-colors-shapes

### Path 3: Addition Introduction
- **Description:** Begin learning addition with small numbers
- **Estimated Time:** 22 minutes
- **Modules:**
  * k-counting-1-20
  * k-comparing-sizes
